Print Index

Win64, English
Ryan McCullough
Save time by automatically generating saved print sets directly from items in a sheet index or sheet list schedule with the click of a button.
General Usage Instructions Screenshots Commands Installation/Uninstallation Contact Version History

General Usage Instructions

  1. First set up your sheet index however you like, you may include filters and sorting of any kind to show the appropriate sheets for the document set you are preparing to print.
  2. When your sheet index accurately reflects the sheets you intend to print/export, click the Print Index app button (note: you do not need to have the sheet index as the active view).
  3. The app will display a list of schedules currently in your model; select the sheet index you intend to print from by clicking on it once.
  4. Next you will need a name to save the View/Sheet set under, type a unique name into the lower text box (e.g. “2014-04-22 MEP Set”).
  5. When you are finished click the OK button and the app will find all sheets referenced in the schedule and save them as a View/Sheet set with your selected name. It will also inform you of the number of sheets you are going to print.
  6. You can now open the print dialog normally, and under Print Range check “Selected Views/Sheets”, your saved set will already be pre-selected.

Installation/Uninstallation

The installer that ran when you downloaded this plug-in from Autodesk Exchange Apps has already installed the plug-in. You may need to restart the Autodesk product to activate the plug-in. To uninstall this plug-in, click Control Panel > Programs > Programs and Features (Windows 7/8/8.1) or Control Panel > Add or Remove Programs (Windows XP), and uninstall as you would any other application from your system.
